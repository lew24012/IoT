Simple ESP8266 Network

This is a fun little project I made using the ESP8266 microcontroller. The "Pinger" allows you to test ping functionality across the mini network hosted by the AP Hub. Each client hosts a webserver that allows the user to enter an IP address to ping. You can also use the serial monitor to verify ping connectivity.

The Web UDP Client allows users to send basic text messages across the network using UDP. Each client hosts a webserver that allows users to select an IP address to send the message to. The client's connection to the AP Hub is configured in the code itself, so if you make any adjustments you'll need to make sure they match on both the Hub and the UDP Client.
